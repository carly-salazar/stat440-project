STAT 440/540 Group Project Starter

How to use
1. Open this folder in RStudio or VS Code.
2. Place your original dataset in data/raw/ (leave example_stub.csv as a template if helpful).
3. Work in reports/ for each milestone, rendering to PDF.
4. Keep raw data immutable; write cleaned data to data/processed/.
5. Put figures and rendered PDFs in outputs/ (handled automatically by Quarto config).

Submission names
- Group_XX_Proposal.pdf
- Group_XX_Milestone2.pdf
- Group_XX_Milestone3.pdf
- Group_XX_Final_Report.pdf
